Humming Studio プライバシーポリシー

最終更新日: 2026年7月12日

合同会社ツナギビト(以下「当社」)は、アプリケーション「Humming Studio」および関連サービス(以下「本サービス」)における利用者情報の取り扱いについて、以下のとおり定めます。

1. 端末内で完結する処理

  • 録音した音声はお使いの端末内にのみ保存されます。
  • 文字起こしは端末内の機械学習モデル(Whisper)で処理され、音声・テキストが外部へ送信されることはありません。
  • AI トピック抽出・トピックプランナーは Apple Intelligence(オンデバイス処理)を使用します。
  • 本アプリはアクセス解析や広告トラッキングを行いません。

2. 外部へ送信される情報

次の場合に限り、情報が当社または第三者のサーバーへ送信されます。

  • 文字起こしモデルの取得: 初回利用時に機械学習モデルを Hugging Face(huggingface.co)からダウンロードします。この通信にモデル取得以外の情報は含まれません。
  • ホスティング機能での公開(ユーザーが明示的に公開操作をした場合のみ): 公開対象の音声ファイル、番組・エピソード情報(タイトル、説明、アートワーク等)が当社サーバーにアップロードされ、RSS フィードを通じてインターネットに公開されます。

3. アカウント情報

ホスティング機能の利用時に、Sign in with Apple を通じて次の情報を取得・保存します。

  • Apple が発行する利用者識別子
  • メールアドレス(Apple の「メールを非公開」機能による転送アドレスを含む)

これらはアカウント管理、サブスクリプションの購読状態の確認、重要なお知らせの送付にのみ使用し、第三者に提供しません。

4. アプリ内課金

サブスクリプション(Humming Studio Pro)の決済は Apple が処理し、当社がクレジットカード情報等の支払い情報を取得することはありません。当社は購読状態(有効/無効)のみを App Store Server API を通じて確認します。

5. マイクへのアクセス

本アプリは録音のためにマイク(および接続されたオーディオインターフェース)を使用します。マイクへのアクセスは録音機能にのみ使用されます。

6. データの削除

アカウントを削除すると、ホスティング中のコンテンツ、アカウント情報は 30 日間の猶予期間の後に削除されます。個別のエピソード・番組はアプリ内からいつでも削除できます。
